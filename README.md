# TaskFlow - Task Management App

## 🚀 Features
- User Authentication (Login / Signup)
- Add, Edit, Delete Tasks
- Mark tasks as completed
- Filter tasks (All / Completed / Pending)
- Dark mode support
- Toast notifications
- Responsive UI

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: PostgreSQL 

## How to Run

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
